## Agents

Proyecto: Aplicación web del tiempo

Agente: Desarrollador experto web con 12 años de experiencia.
Objetivo: Crear una aplicación web que nos permita ver el tiempo de cualquier ciudad del mundo de manera sencilla y rápida, consumiendo un API rest externa y dando la posibilidad de ver el tiempo actual y el pronóstico del tiempo para las próximas 12 horas y dando la posibilidad de guardar las diferentes ciudades en el local storage para poder verlas de manera rápida.

URL del API (no requiere autenticación, ni API key): https://open-meteo.com/

Funcionalidades:
- Búsqueda de la ciudad.
  - Input para buscar la ciudad.
  - Botón para buscar la ciudad.
  - Validación básica del campo.
  - Dar sugerencias de ciudades mientras se escribe.
- Clima actual:
  - Nombre de la ciudad
  - Temperatura actual
  - Descripción del clima (Soleado, nublado, lluvioso, etc)
  - Icono representativo del clima actual
  - Sensación térmica
  - Humedad
  - Velocidad del tiempo
  - Botón para guardar la ciudad
- Localidades guardadas
  - Boton para guardar la ciudad.
  - Boton para eliminar la ciudad.
  - Ver un listado de ciudades guardadas.
  - Evitar guardar duplicados. 

Stack tecnológico:
- HTML5
- CSS3
- JavaScript Vanilla

Preferencias de diseño:
- Basate en las imagenes de referencia para el diseño y en el HTML del diseño, se encuentran en la carpeta "designs".
- Responsive
- Minimalista
- Moderno
- Claro

Preferencias de estilo:
- Eliminar Tailwind CSS y pasarlo a CSS puro.
- Colores: Usa los colores de las imagenes de referencia.
- No uses tailwind, ni bootstrap, ni ningun framework de css.
- Uso de medidas con rem, usando font size de base de 10 px.
- Que la app esté en español e inglés.
- Que la app sea responsive.
- Usa buenas practicas de maquetación css y usa flexbox si fuese necesario y css grid layout. 

Preferencias de código:
- No añadas dependencias externas.
- HTML semántico.
